Template Description

Spruha is a React Admin & Dashboard template using modern and minimal design. It is fully flexible user-friendly and responsive. Spruha React-Admin template is powered with React, SASS, & React-Bootstrap Framework which looks great on Desktops, Tablets, and Mobile Devices. React-Bootstrap replaces the Bootstrap JavaScript. Each component has been built from scratch as a true React component, without unneeded dependencies like jQuery. This Template Includes 100+ React Pages & 40+ Plugins, Material UI & more UI elements .
